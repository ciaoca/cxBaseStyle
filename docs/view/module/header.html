Style
<div class="header">
  <header>
    <h1></h1>
  </header>
</div>

.notfixed

鉴于移动端的特性，头部默认为 fixed，需要非固定时再添加。

.transparent

适用于页面顶部有背景时，可将头部背景去除。（另可根据 scrollTop 距离切换该样式，避免在页面滚动后，头部无背景难以识别。）

.darktext

若顶部背景为亮色调时，可将文字调整为暗色调。

.notsize

适用于顶部背景的元素设在头部元素之后，去除头部的高度，让该元素到达顶部。

添加操作区域
<div class="header">
  <header>
    <h1></h1>
    <div class="group onleft">
      <a class="icon back" href="index.html"></a>
    </div>
    <div class="group onright">
      <a class="icon go" href="login.html"></a>
    </div>
  </header>
</div>

增加div.group标签，通过.toleft,.toright设置位置。

内部可添加a标签，或其他内容。

a 标签支持样式：

class="txt"纯文字按钮

class="arrow back"带左箭头的文字按钮

class="arrow go"带右箭头的文字按钮

class="icon back"后退按钮

class="icon go"前进按钮

class="icon close"关闭按钮

class="icon add"加号按钮

class="icon *"其他自定义按钮
